Projects

➢ The monitor system of resistance spot welding                                                                       Jan.2018-May.2019

  • Use the STM32 to monitor welding parameters. GUI is designed by LabVIEW to show all necessary curves, set parameters of the monitor and store all data into a database.

  • Program the FPGA and chip AD7768 to sampling data (8-channel 24-bit data sampling system).

  • Detect welding expulsion and evaluate welding quality depending on dynamic resistance curve. Use machine learning to recognize the material combination for RSW. Monitor the electrode wear state base on time series similarity of dynamic resistance.

 


➢ SMA-actuated steerable neurosurgical robot                                                                           Jul.2019-Aug.2019

  • Design experiments to characterize the SMA spring. Use the EM-Tracker to track the position of the steerable end effector.

  • Implement a cascade PI controller with feedforward compensation to actuate the robot.
